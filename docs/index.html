UC Verifiable Credentials

Demonstration of a blockchain-backed verification system for University of California documents. Register document fingerprints, confirm authenticity instantly, and keep records tamper-proof without uploading personal data.

How It Works

  1. Admins connect an Ethereum wallet on Sepolia to issue credentials.
  2. Students request documents off-chain; admins issue on-chain when approved.
  3. Anyone can verify by uploading a file or supplying a hash.

Why Blockchain?

We store only SHA-256 hashes of documents on-chain, creating immutable proofs of existence. Anyone with the original file can recompute its fingerprint and verify ownership, while the contents remain private.
